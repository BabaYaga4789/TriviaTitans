## Run Locally

Clone the project

```bash
  git clone https://git.cs.dal.ca/vikramv/csci5410-summer-23-sdp17.git
```
Checkout to "#7_Trvia_Content_Management"

```bash
  cd tutorial3
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
